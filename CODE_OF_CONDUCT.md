# CrazyUX – Code of Conduct

## 1. Our Commitment
The **CrazyUX community**, maintained by **QuCodeXClub**, is committed to providing a welcoming, inclusive, and respectful environment for everyone. 
We encourage collaboration, creativity, and learning among developers, designers, and contributors participating in this project.

We aim to ensure that participation in this project remains a **positive experience for all contributors**, regardless of background, experience level, gender, identity, or nationality.

---

## 2. Expected Behavior
All contributors are expected to:

- Be respectful and considerate toward other contributors.
- Encourage collaboration, learning, and knowledge sharing.
- Provide constructive feedback in discussions and pull requests.
- Maintain professionalism in issues, discussions, and commits.
- Follow ethical open-source practices.
- Respect different viewpoints and experiences.

---

## 3. Unacceptable Behavior
The following behaviors are strictly prohibited within the project community:

- Harassment, discrimination, or offensive comments.
- Personal attacks, trolling, or insulting language.
- Posting inappropriate, illegal, or harmful content.
- Sharing private information of others without permission.
- Spamming issues, pull requests, or discussions.
- Submitting plagiarized or stolen code without attribution.

---

## 4. Scope
This Code of Conduct applies to all spaces associated with the CrazyUX project, including:

- GitHub Issues
- Pull Requests
- Discussions
- Project documentation
- Any official communication channels related to the project

---

## 5. Reporting Issues
If you experience or witness unacceptable behavior:

1. Report the issue to the project maintainers.
2. Provide clear details and, if possible, supporting evidence.
3. All reports will be handled confidentially and respectfully.

---

## 6. Enforcement
Project maintainers have the authority to take appropriate actions in response to violations of this Code of Conduct, including:

- Issuing warnings
- Removing inappropriate content
- Restricting participation in the project
- Temporarily or permanently banning contributors

---

## 7. Acknowledgment
By contributing to or participating in this project, you agree to follow this Code of Conduct and help maintain a respectful and collaborative open-source community.
